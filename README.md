# android-log.vim

A simple syntax highlight plugin for Android logs.

## Usage

In Android log files, enter `:set ft=android_log` can open it.

File types below are supported automatically:

- *.alog
- main_log
- main_log.\d*
- sys_log
- sys_log.\d*
- radio_log
- radio_log.\d*
- crash_log
- events_log
